# AN DESIGN SYSTEM: DUAL IDENTITY

## 1. TYPOGRAPHY (SHARED DNA)
* **Font Family:** `Inter` (Google Fonts).
* **Usage:**
    * *Connect Mode:* Headings are Medium/SemiBold. Body is Regular.
    * *Career Mode:* Headings are Bold/Black. Uppercase allowed for labels.

## 2. THEME A: "AN CONNECT" (Product / User / Healing)
* **Context:** Assessment Pages, User Dashboard, Profile.
* **Primary Gradient:** `linear-gradient(90deg, #189cab 0%, #8bc540 100%)` (Teal to Lime).
* **Primary Color:** `#189cab` (Teal).
* **Secondary:** `#8bc540` (Lime).
* **Radius (Bo góc):** `1.5rem` (24px) - "Super Rounded".
* **Shadow:** `0 10px 30px -10px rgba(24, 156, 171, 0.2)`.
* **Vibe:** Soft, Organic, Safe.

## 3. THEME B: "AN CAREER" (Internal / Data / Professional)
* **Context:** Recruitment Dashboard, Analytics Matrix, Admin Panel.
* **Primary Color:** `#4285F4` (Google Blue).
* **Semantic Colors:**
    * Error/Risk: `#EA4335` (Red).
    * Success/KPI: `#34A853` (Green).
    * Warning/Idea: `#FBBC04` (Yellow).
* **Radius (Bo góc):** `0.375rem` (6px) - "Structured".
* **Background:** `#F1F3F4` (Grid Paper Style).
* **Vibe:** Data-driven, Efficient, Strict.

## 4. LOGO VARIATIONS
* **Connect Logo:** Filled with Teal-Lime Gradient.
* **Career Logo:** Filled with Solid Google Blue.
