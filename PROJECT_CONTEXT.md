## 🎯 MỤC ĐÍCH ỨNG DỤNG (PURPOSE & MISSION)

### Vấn đề cốt lõi (Pain Point)
- **"The Service Cliff" (Vực thẳm Dịch vụ):** Tại Việt Nam, thanh thiếu niên tự kỷ sau 18 tuổi mất hầu hết hỗ trợ can thiệp sớm, rơi vào tình trạng thất nghiệp dài hạn.
- **Hệ thống tuyển dụng lỗi thời:** Phỏng vấn truyền thống đánh giá kỹ năng giao tiếp hơn là năng lực thực tế → loại bỏ người Đa dạng thần kinh dù họ có thể làm việc xuất sắc.

### Giải pháp
**An không hỏi "Bạn có giỏi không?" mà chứng minh "Bạn làm được việc gì?"**

1. **Game-Based Assessment (Đánh giá qua chơi game):** 
   - Thay thế phỏng vấn bằng các trò chơi khoa học
   - Thu thập dữ liệu Telemetry (Reaction time, Accuracy, Cognitive patterns)

2. **Digital Career Passport (Hồ sơ nghề nghiệp số):**
   - Chuyển hóa dữ liệu game thành chứng chỉ năng lực
   - Hiển thị điểm mạnh qua Radar Chart (Biểu đồ mạng nhện)

3. **AI-Powered Job Matching (Khớp việc làm bằng AI):**
   - Vector Matching: So sánh profile người dùng với yêu cầu công việc
   - Đưa ra Top 3 nghề nghiệp phù hợp nhất + giải thích chi tiết

### Mục tiêu cuối cùng
- **Con số:** Giảm 50% tỷ lệ thất nghiệp trong nhóm Đa dạng thần kinh 18-25 tuổi tại VN
- **Tác động xã hội:** Thay đổi cách xã hội đánh giá năng lực (từ CV/bằng cấp → năng lực thực tế)

## ⚙️ TÍNH NĂNG CHÍNH (CORE FEATURES)

### 1. **HỆ THỐNG ĐÁNH GIÁ 3 TRÒ CHƠI (Game-Based Assessment Engine)**

#### Game 1: Detail Spotter (Săn lỗi hình ảnh)
- **Lý thuyết:** Visual-Spatial Intelligence (Đa trí tuệ Gardner)
- **Cơ chế:** 
  - Hiển thị lưới 3x3 hình ảnh (Landolt C, Mirror Image)
  - Người chơi phải tìm hình khác biệt trong thời gian giới hạn
- **Đo lường:**
  - `visual_acuity`: Độ chính xác nhận diện (%)
  - `reaction_time_ms`: Thời gian phản xạ (mili giây)
  - `error_rate`: Tỷ lệ nhầm lẫn
- **Nghề nghiệp liên quan:** Data Labeling (Gán nhãn AI), QC Inspector, Graphic Designer

#### Game 2: Chaos Switcher (Luật đổi ngẫu nhiên)
- **Lý thuyết:** Logical-Mathematical Intelligence + Stroop Effect
- **Cơ chế:**
  - Màu sắc và chữ viết không khớp (Xung đột thần kinh học)
  - Luật chơi thay đổi bất ngờ (Ví dụ: Nhấn màu → Nhấn chữ)
- **Đo lường:**
  - `cognitive_flexibility`: Khả năng thích nghi (điểm 0-100)
  - `inhibition_control`: Ức chế xung động (số lần sai khi đổi luật)
  - `switch_cost`: Chi phí thời gian khi đổi luật
- **Nghề nghiệp liên quan:** Logistics Coordinator, Software Tester, Dispatcher

#### Game 3: Sequence Master Pro (Trí nhớ chuỗi đảo ngược)
- **Lý thuyết:** Working Memory Capacity (Baddeley Model)
- **Cơ chế:**
  - Nhớ chuỗi số/hình → Nhập lại theo thứ tự đảo ngược
  - Có nhiễu (âm thanh/hình ảnh gây xao lãng)
- **Đo lường:**
  - `memory_span`: Độ dài chuỗi tối đa (số phần tử)
  - `working_memory_score`: Điểm trí nhớ làm việc
  - `distraction_resistance`: Khả năng kháng nhiễu
- **Nghề nghiệp liên quan:** Programmer, Assembly Technician, Chef (quy trình phức tạp)

### 2. **AI CAREER ADVICE (Tư vấn nghề nghiệp bằng AI)**

#### Công nghệ
- **Model:** OpenRouter API (tngtech/tng-r1t-chimera:free)
- **Input:** Điểm số 4 chỉ số (Visual, Logic, Memory, Speed) + Top 3 nghề nghiệp
- **Output:** 
  - Lời khuyên cá nhân hóa (3-4 câu)
  - Giải thích vì sao phù hợp với nghề X
  - Gợi ý đào tạo cụ thể

#### Edge Function
- **Service:** `generate-advice` (Supabase Edge Function)
- **Logic:** Gọi OpenRouter API với prompt template có sẵn
- **Caching:** Không cache (để đảm bảo lời khuyên luôn mới)

### 3. **DIGITAL CAREER PASSPORT (Hồ sơ năng lực số)**

#### Thành phần
1. **Radar Chart (Biểu đồ mạng nhện 4 chiều):**
   - Visual Acuity (Thị giác)
   - Logical Thinking (Tư duy logic)
   - Working Memory (Trí nhớ)
   - Processing Speed (Tốc độ xử lý)

2. **Tier System (Xếp hạng 3 cấp):**
   - **Tier 1 - Elite:** Top 5% dân số → High-Tech Jobs (AI Trainer, Coder)
   - **Tier 2 - Professional:** Top 20% → Standard Jobs (Admin, QC)
   - **Tier 3 - Potential:** Cần đào tạo thêm → Entry-level

3. **Job Match Cards (Thẻ nghề nghiệp):**
   - Top 3 nghề phù hợp nhất
   - % Match Score (Ví dụ: "98% phù hợp với Data Labeler")
   - Icon + màu sắc phân biệt từng ngành

### 4. **VECTOR JOB MATCHING (Khớp việc làm thông minh)**

#### Database nghề nghiệp
20+ nghề nghiệp được định nghĩa trước, mỗi nghề có vector yêu cầu:
```typescript
{
  title: "Coder (Lập trình viên)",
  requirements: {
    visual: 0.3,   // 30% quan trọng
    logic: 0.8,    // 80% quan trọng (chỉ số chính)
    memory: 0.9,   // 90% quan trọng (chỉ số chính)
    speed: 0.4     // 40% quan trọng
  }
}
```

#### Thuật toán khớp
1. **Chuẩn hóa điểm người dùng:** 0-100 → 0-1
2. **Tính Cosine Similarity:** So sánh vector người dùng vs vector nghề nghiệp
3. **Sắp xếp:** Top 3 nghề có similarity cao nhất
4. **Giải thích:** AI tạo lời khuyên dựa trên chỉ số nổi bật nhất

## 🔧 VAI TRÒ CỦA TỪNG TÍNH NĂNG (FEATURE ROLES)

## 🛑 AUDIT REPORT SUMMARY (V3 - 2026-01-19)
**Auditor:** Antigravity Agent
**Status:** 🔴 NOT READY FOR FUNDING (Critical Issues Found)

### Key Findings
1. **Science (Broken):** The "Global Comparison" (Z-Score) is completely broken by RLS policies `auth.uid() = user_id`, meaning users are compared only to themselves. `AssessmentResult` hardcodes 3/5 metrics (`memory: 65`, etc.).
2. **Business (Non-Existent):** There is no Employer Dashboard or code to support B2B logic. It's purely a B2C toy.
3. **Market (Dangerous):** The `job_crawler.py` fails to filter out "Dynamic/High Pressure" keywords, risking sending autistic users into meltdown-inducing jobs. It relies on a fragile scraper for TopCV.

### The Fix List (Top 5 Priorities)
1. **FIX THE BRAIN:** Create a `global_stats_view` (Security Definer) to allow anonymous aggregate stats reading for valid Z-Scores. Remove hardcoded metrics.
2. **SAFETY FIRST:** Update Crawler to penalize jobs with "Dynamic/Sales" keywords (`NEGATIVE_KEYWORDS`).
3. **LEGAL:** Add a "Not a Medical Diagnosis" disclaimer checkbox.
4. **RESILIENCE:** Decouple from TopCV scraping; build a simple internal Job Board table.
5. **VISION:** Build a "Mock" Employer Dashboard to demonstrate the business value to investors.
